# Órbita Galáctica

Un emocionante juego de disparos espacial online con sistema de niveles, experiencia y personalización de naves.

## Características
- Sistema de niveles y XP persistente.
- Personalización de naves en el hangar.
- Tienda de módulos y munición especial.
- Diferentes tipos de naves: Tanque, Rápida, Sigilo, Pesada y Soporte.
- Combate en tiempo real contra aliens.
